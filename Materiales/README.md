# Materiales de clase

Esta carpeta contiene material preparado para impartir el curso.

Cada clase tiene dos tipos de archivo:

- `.html`: fuente editable del documento.
- `.pdf`: version lista para compartir o imprimir.

## Clase 1

- `clase-01-entorno-y-hola-mundo.html`
- `Clase_01_Entorno_y_Hola_Mundo.pdf`

Contenido:

- instalacion de Visual Studio Code,
- extensiones recomendadas,
- cuenta de GitHub,
- configuracion de Git,
- conexion con GitHub y VS Code,
- instalacion de Node.js,
- uso basico de terminal,
- creacion de un primer proyecto React con Vite,
- primer Hola Mundo,
- ejercicios.

## Clase 2

- `clase-02-stack-rutas-y-contacto.html`
- `Clase_02_Stack_Rutas_y_Contacto.pdf`

Contenido:

- repaso del proyecto React,
- estructura del stack,
- orden de trabajo paso a paso para crear el stack,
- carpeta `Backend/`,
- dependencias base y dependencia adicional `react-router-dom`,
- comandos de consola necesarios,
- instalacion de React Router,
- rutas internas,
- pagina `Inicio`,
- pagina `Contacto`,
- navegacion con `Link`,
- componentes sencillos `Texto` y `Boton`,
- explicacion de `eslint.config.js`,
- codigo de los archivos principales del stack,
- flujo `index.html -> main.jsx -> App.jsx -> rutas -> paginas`,
- teoria JavaScript/JSX usada en el stack,
- ejercicios.
